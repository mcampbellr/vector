# Refiner base rules

## Preserve the user's language

Write the output in the same language as the input. Spanish input → Spanish output. English
input → English output. Kebab-case identifiers (change names, slugs) are always in English
regardless of the input language.

## Be terse

Each output section is the minimum useful content. No filler, no restating the user's input
verbatim. Every sentence must add information the caller does not already have.
